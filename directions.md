# 使用说明
```
脚本启动时，会检查是否以管理员权限运行。
如果不是管理员权限，会弹出警告框提示用户以管理员权限运行。
虽然依旧可以打开脚本，但有可能(其实是百分百)会出问题，建议以管理员权限打开脚本。
```
## 基本操作
- 点击左侧导航栏的按钮可以切换到不同的页面。

### 任务页面
- 左侧为当前的任务列表，点击右侧面板最上方的“添加任务”按钮可以添加新的任务。
    - 列表支持右键菜单，可调整位置/删除任务，也可以直接点击列表下方的清空按钮一键清空任务列表
- 右侧面板可以设置任务的详细参数，包括窗口名称、以及各种参数
    - 窗口名称：重要参数，用于获取窗口句柄，默认为"一梦江湖"，基本不用改。
    - 循环次数：设置脚本运行的次数，默认值为1次。
    - 图片匹配阈值：设置图片匹配的阈值，范围为0~1，默认值为0.6。
    - 间隔时间
        - 点击间隔：设置每次点击之间的间隔时间，单位为秒，默认值为1秒。
        - 匹配间隔：设置每次匹配图片之间的间隔时间，单位为秒，默认值为0.5秒。
        - 截图重试时间：设置截图失败后的重试时间，单位为秒，默认值为3秒。
        - 截图重试次数：设置截图失败后的重试次数，默认值为3次。
        - 超时时间：设置任务线程的超时时间，单位为秒，默认值为600秒。
- 点击右侧面板最上的“运行任务”按钮可以依次运行任务列表中的任务。
- 点击右侧面板最上的“停止任务”按钮可以停止当前运行的任务。
- 点击右侧面板的“显示窗口”可以显示当前绑定的窗口。
- 开始运行前需先绑定窗口，点击右侧面板的“绑定窗口”按钮可以绑定指定名称的窗口。

## 使用方法
1. 添加任务
    - 点击左侧导航栏的“任务”按钮切换到任务页面。
    - 点击右侧面板最上方的“添加任务”按钮添加新的任务。
    - 任务添加后，会在任务列表中显示出来。
    - 可以自由调整顺序。
2. 设置任务参数
    - 点击右侧面板最上方的“任务配置”按钮。
    - 目前仅支持一键设置所有任务的参数，无法单独设置。
    - 根据需要修改参数值，确保参数设置符合要求。
3. 运行任务
    - 点击右侧面板最上方的“运行任务”按钮开始运行任务。
    - 任务运行过程中，可点击“停止任务”按钮随时停止运行。
4. 没了

## 注意事项
- 此脚本仅支持Windows系统。
- 请确保在运行脚本前，已经打开了目标窗口，并且窗口名称与设置的窗口名称一致。
- 脚本运行期间可以操作其它窗口，但不要将游戏窗口最小化。
- 脚本运行过程中，不要关闭目标窗口，否则会导致脚本运行异常。
- 游戏窗口的大小最好保持在1600x900左右，实测这个大小的窗口匹配效果较好，1200x600左右也行，别太大也别太小。
- 目前仅支持日常任务和论剑，别的还没写。
